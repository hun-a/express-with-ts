# What's this?
Tutorial for the Express with Typescript!

# How to run?
### prerequisite
- Create the `.env` file in the root directory fo this project.
- Example of `.env`
    ```
    MONGO_USER=dbUser
    MONGO_PASSWORD=dbUserPassword
    MONGO_PATH=@ds111111.mlab.com:27594/test
    PORT=5000
    ```

#### Run in DEV environment
```
$ npm run dev
```
